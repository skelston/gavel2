# Gavel 2

**Gavel** is a powerful, elegant tool for judging project expos at hackathons and other events. It uses a proven, mathematically sound algorithm to efficiently identify the top projects by comparing them in a series of head-to-head votes, even with a large number of entries and a limited number of judges.

This repository, **Gavel 2**, is a maintained and updated fork of the original system.

### Lineage & History: A Clearer Picture

The evolution of Gavel can be a bit complex. Here’s the straightforward breakdown of how this fork came to be:

1.  **Original Gavel (by [anishathalye](https://github.com/anishathalye/gavel))**: The classic, battle-tested system first used at HackMIT 2015. Dozens of events have used it since.
2.  **Gavel 2 (by [weareasterisk](https://github.com/weareasterisk/gavel2))**: A community fork featuring a significantly improved, more modern user interface (UI) and user experience (UX).
3.  **Hib4's Fork (of Gavel 2)**: A personal fork that updated the codebase for GarudaHacks 2024.
4.  **This Fork (by [skelston](https://github.com/skelston/gavel2))**: A parsonal effort to revive and modernize the codebase. It's based directly on `hib4/gavel2` (which is based on `weareasterisk/gavel2`) and is focused on **making it work** with current dependencies, fixing bugs, and ensuring it can be easily deployed for events today.
